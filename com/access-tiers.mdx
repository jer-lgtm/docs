## Tier Upgrade Process

### Automatic Upgrades

<Steps>
<Step title="Stake Additional Tokens">
  Add more COM tokens to your existing stake to reach the next tier threshold.
  
  ```bash
  # Example: Upgrade from Basic to Standard
  current_stake = 100 COM
  additional_needed = 400 COM
  comput3 stake --amount 400
  ```
</Step>

<Step title="Instant Activation">
  Tier benefits activate immediately upon reaching the staking threshold.
  
  <Check>
  No waiting period - start using enhanced features right away.
  </Check>
</Step>

<Step title="Retroactive Credits">
  Credit generation rate adjusts automatically with bonus applied to current day.
  
  **Example**: Upgrade at noon → receive bonus credits for remaining day
</Step>
</Steps>

### Tier Downgrade

<Warning>
Reducing your stake below a tier threshold immediately downgrades your access level.
</Warning>

**Downgrade Process**:
1. Unstake tokens below tier threshold
2. Tier benefits immediately deactivate  
3. Credit generation rate adjusts to new tier
4. Existing credits remain usable
