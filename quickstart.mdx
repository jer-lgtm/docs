## Get started in three steps

Connect to the decentralized GPU network and run your first workload.

### Step 1: Set up your account

<AccordionGroup>
  <Accordion icon="user-plus" title="Create your Comput3 account">
    1. Visit [app.comput3.ai](https://app.comput3.ai) to create your account
    2. Complete the verification process
    3. Fund your account with credits or connect your wallet
    
    <Tip>New users receive 100 free compute credits to get started!</Tip>
  </Accordion>
  <Accordion icon="key" title="Get your API credentials">
    1. Navigate to your [dashboard](https://app.comput3.ai/dashboard)
    2. Go to Settings â†’ API Keys
    3. Generate your API key and save it securely
    
    ```bash
    export COMPUT3_API_KEY="your-api-key-here"
    ```
    
    <Warning>Never share your API key or commit it to version control.</Warning>
  </Accordion>
</AccordionGroup>

### Step 2: Install the Comput3 CLI

<AccordionGroup>
  <Accordion icon="download" title="Install via npm">
    ```bash
    npm install -g @comput3/cli
    ```
    
    Or using pip:
    ```bash
    pip install comput3-cli
    ```
  </Accordion>
  <Accordion icon="terminal" title="Verify installation">
    ```bash
    comput3 --version
    comput3 auth login
    ```
    
    <Tip>The CLI will automatically use your COMPUT3_API_KEY environment variable.</Tip>
  </Accordion>
</AccordionGroup>

### Step 3: Run your first GPU workload

<Accordion icon="rocket" title="Deploy a sample job">
  ```bash
  # Run a simple GPU test
  comput3 run --gpu-type A100 --replicas 1 \
    --image pytorch/pytorch:latest \
    --command "python -c 'import torch; print(torch.cuda.is_available())'"
  
  # Monitor your job
  comput3 jobs list
  comput3 logs <job-id>
  ```
  
  <Success>Congratulations! You've just run your first job on the Comput3 network.</Success>
</Accordion>

## SDK Examples

<CodeGroup>
```python Python
from comput3 import Client

client = Client(api_key="your-api-key")

# Submit a job
job = client.jobs.create(
    gpu_type="A100",
    replicas=2,
    image="tensorflow/tensorflow:latest-gpu",
    command="python train.py"
)

# Monitor progress
status = client.jobs.get(job.id)
print(f"Job {job.id}: {status.state}")
```

```javascript Node.js
const { Comput3Client } = require('@comput3/sdk');

const client = new Comput3Client({ 
  apiKey: 'your-api-key' 
});

// Submit a job
const job = await client.jobs.create({
  gpuType: 'A100',
  replicas: 2,
  image: 'tensorflow/tensorflow:latest-gpu',
  command: 'python train.py'
});

// Monitor progress
const status = await client.jobs.get(job.id);
console.log(`Job ${job.id}: ${status.state}`);
```
</CodeGroup>
