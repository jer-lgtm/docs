---
title: 'Create Compute Job'
openapi: 'POST /jobs'
---

Create a new GPU compute job on the Comput3 Network.

## Authentication Required

This endpoint requires a valid API key in the Authorization header:

```bash
Authorization: Bearer YOUR_API_KEY
```

## Example Request

```bash curl
curl -X POST "https://api.comput3.ai/v1/jobs" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "deepseek-v3.1",
    "prompt": "Hello, world!",
    "max_tokens": 100,
    "temperature": 0.7
  }'
```

## Request Body

- `model` (string, required): The model ID to use for computation (e.g., "deepseek-v3.1", "hermes4:70b", "qwen3-coder:480b")
- `prompt` (string, required): Input prompt for the model
- `max_tokens` (integer): Maximum number of tokens to generate
- `temperature` (float): Sampling temperature (0.0 to 1.0)

## Response

Returns the created job with its unique identifier and status.

```json
{
  "id": "job_123abc",
  "status": "queued",
  "model": "deepseek-v3.1",
  "created_at": "2024-01-01T00:00:00Z"
}
```
